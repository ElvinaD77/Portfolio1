Pengertian HTML

HyperText Markup Languange (HTML) adalah bahasa markah standar untuk dokumen yang dirancang untuk ditampilkan diperamban intarnet. Ini dapat dibantu oleh teknologi seperti Cascading Style Sheets dan bahasa skrip lainnya seperti JavaScript, VBScript, dan PHP.





Pengertian CSS

Cascading Style Sheet (CSS) merupakan aturan untuk mengatur beberapa kompenen dalam sebuah web, sehingga akan lebih terstruktur dan seragam. CSS bukalah sebuah bahasa pemrograman.
